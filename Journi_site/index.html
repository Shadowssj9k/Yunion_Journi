HTML


What Does HTML Stand For?

HyperText Markup Language.

What is HTML used for?

Creating the structure of a webpage, with building blocks known as Elements.

What is an Element?

WAn element represents an individual component of an HTML document, like an image or button These elements are represented with tags.

What is an Attribute?

An Attribute describes more information about an element. Attributes are set in the opening tag of every element.
